Memory map#

The following table describes the memory layout of the ByteCradle 6502 SBC:

Address Range

Description

$0000-$00FF

Zero Page

$0100-$01FF

Stack

$0200-$02FF

Keyboard Buffer

$0300-$03FF

System Variables

$0400-$7EFF

Free User Space

$7F00-$7FFF

I/O Space

$8000-$BFFF

Bankable Memory (32 x 16 KiB)

$C000-$FFFF

Bankable System ROM (32 x 16 KiB)

Zero Page#

The first 32 bytes of the zero page are reserved for system variables. The user is free to use the zero page at $20-$FF.
